# 코스트코 CE/MX 실적 자동 취합 웹앱 v5

## 변경 사항
- 보고용_요약 시트에서 품목별 상세 컬럼 제거
- 총 목표 → 총 실적 → 총 달성률 → CE → MX 순서로 재정렬
- 담당자명 K/D 제거
- 담당별 총 매출 행 스타일 구분
- 점포별 총 매출/CE 매출/CE 달성률/총 달성률/MX 매출/MX 달성률 순위 추가
- 보고멘트 시트 추가
- 모델_TOP 시트 추가
- SheetJS 스타일 방식 대신 ExcelJS 생성 방식 적용
