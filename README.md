# 코스트코 CE/MX 실적 자동 취합 웹앱 v7

## 변경 사항
- 에어컨 세트 기준표 선택 업로드 추가
- AF60F17D11WT / AF60F17D11WRT / AF80F17D21CT / AF80F19D24CRT / AR60F07D12WT / AW05C7155WWAZ 등 세트명 기준 집계 지원
- 상품DB 구조가 `세트명 행 → 구성품 행`으로 되어 있으면 구성품을 상단 세트명으로 자동 매핑
- 구성품이 여러 세트명에 중복 매핑되면 검증로그에 `에어컨 세트 중복구성품`으로 표시
- 기존 보고용_요약, 매출 요약본, 보고멘트, 에어컨_세트별 시트 유지

## 사용법
1. 제니엘 실적 데이터 업로드
2. 코스트코 MX SI 실적 업로드
3. 26.5월 점별 목표 업로드
4. 필요 시 에어컨 세트 기준표 업로드
5. 기준월 확인
6. 보고용 엑셀 생성 클릭

## 에어컨 세트 기준표 권장 구조
- A열: 모델명
- B열: 세트명/구성품 설명
- 같은 세트의 구성품은 세트명 행 바로 아래에 위치

예시:
- AF60F17D11WT / 스탠드 단품
- AF60F17D11WN / 에어컨 스탠드 명
- AF60F17D1QBX / 에어컨 실외기 명
- AF60F17D11WRT / 스탠드+벽걸이
- ...
