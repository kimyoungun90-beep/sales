# 코스트코 CE/MX 실적 자동 취합 웹앱

## 사용 방법
1. `index.html`을 브라우저에서 엽니다.
2. 아래 3개 파일을 업로드합니다.
   - 제니엘 실적 데이터
   - 코스트코 MX SI 실적
   - 26.5월 제니엘 점별 목표
3. MX 기준월을 선택합니다. 기본값은 `2026-05`입니다.
4. `보고용 엑셀 생성` 버튼을 누릅니다.

## 생성 시트
- 보고용_요약: 담당/점포별 CE, MX, 전체 목표·실적·달성률 및 품목별 실적
- CE_품목별: 제니엘 기준 CE 품목별 실적
- MX_품목별: MX SI 기준 휴대폰, 태블릿, PC, 웨어러블 실적
- 목표_확인: 목표 파일에서 가져온 점포별 CE/MX 목표
- 검증로그: 반영/제외 건수, 기준월 컬럼, 일자 범위 등 확인용

## 적용 기준
- 제니엘: C열 `코스트코`, E열 점포, F열 품목, H열 `금액`, I열 이후 일자별 금액 합산
- 제외 점포: 평택물류창고, CDC온라인창고, 코스트코 코리아
- 제니엘에서 제외하는 MX 품목: 웨어러블, PC, 모바일, 태블릿, 스마트폰, 휴대폰, 자급제폰
- MX SI: `매장별 셀인_26년(금액)` 시트 8행에서 선택 월 컬럼 자동 탐색
- 목표: `매장별 목표` 시트의 `CC` 행만 사용
