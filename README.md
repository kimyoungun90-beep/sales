- 에어컨 집계를 세트명 분리 방식에서 월 단위 모델군 방식으로 변경
- Q9000 17평 / Q9000 19평 / 클래식 17평 / 갤러리 17평 / 갤러리 19평 / 벽걸이 / 창문형 구분 가능
- 공용 자재·설치비·리모컨·벽걸이 금액은 스탠드/본품 판매수량 비율로 배분
- `에어컨_모델군` 시트 추가
- `에어컨_배분로그` 시트 추가
- 기존 보고용_요약, 보고멘트, CE_품목별, MX_품목별, 모델_TOP, 목표_확인, 검증로그 유지

## 에어컨 월 모델군 기준
- 제니엘 파일의 일자별 컬럼은 모두 월 합산 처리
- 스탠드/본품 수량을 판매수량 기준으로 사용
- 한 모델군에만 들어가는 구성품 금액은 해당 모델군에 직접 반영
- 여러 모델군에 공용으로 들어가는 구성품 금액은 해당 월 점포별 스탠드/본품 판매수량 비율로 배분
