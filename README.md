# 코스트코 CE/MX 실적 자동 취합 웹앱 v6

## 변경 사항
- 보고용_요약 시트 4행에 총 매출 / CE 매출 / MX 매출 대분류 헤더 추가
- 각 영역을 목표 / 매출 / 순위 / 달성률 / 순위 순서로 재정렬
- 웹앱 화면에 20개 점포 매출 요약본 미리보기 추가
- 요약본 표 복사 버튼 추가
- 에어컨 모델을 세트명 기준으로 표기하도록 보정
- 에어컨_세트별 시트 추가: 세트명 / 판매수량 / 매출 확인용
- 보고멘트와 모델_TOP 시트의 에어컨 대표 모델도 세트명 기준 반영
